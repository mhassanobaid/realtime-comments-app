# Real-Time Comments App (Rails + ActionCable)

A simple demonstration project showing real-time updates using WebSockets.
Users can post comments and all connected clients instantly receive updates without page refresh.

## Tech Used
- Ruby on Rails
- ActionCable (WebSockets)
- Turbo / Importmap / JS
- SQLite

## Clone Project
- git clone https://github.com/mhassanobaid/realtime-comments-app.git
- cd realtime-comments-app
- bundle install
- rails db:create db:migrate
- rails s
